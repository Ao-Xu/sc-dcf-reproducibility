# SC-DCF reproducibility package

This repository contains the code and generated outputs for the computational
experiments in the manuscript:

**Scalable Multi-Output Lipschitz Regression via Shared-Center Delta-Convex
Fitting**

The experiments evaluate the least-squares implementation `SC-DCF-LS` against
independent DCF-LS and standard multi-output baselines. The script regenerates
the CSV summaries, LaTeX tables, and figures used in the manuscript.

## Contents

- `run_experiments.py`: main experiment and table/figure generation script.
- `run_all_experiments.ps1`: Windows PowerShell wrapper for a full rerun.
- `requirements.txt`: Python package versions used for the reported run.
- `REPRODUCIBILITY.md`: concise reproduction notes from the manuscript source.
- `outputs/`: generated CSV summaries, LaTeX tables, and PDF/PNG figures.

Cached downloaded data are not committed. When the script is run, public data
sets are downloaded as needed and stored under `data_cache/`.

## Environment

The reported run used:

- Python 3.7.0
- NumPy 1.21.6
- SciPy 1.1.0
- scikit-learn 1.0.2
- pandas 1.1.5
- matplotlib
- openpyxl

Install dependencies with:

```bash
python -m pip install -r requirements.txt
```

For a fresh modern environment, Python 3.7-compatible packages may require an
older Python environment or a Conda environment.

## Reproduce all outputs

On Windows PowerShell:

```powershell
.\run_all_experiments.ps1
```

Or directly:

```bash
python run_experiments.py
```

The script writes all generated files to `outputs/`. Existing files with the
same names are overwritten.

## Public data sources

The script downloads only public benchmark data:

- UCI Energy Efficiency data set.
- UCI Communities and Crime Unnormalized data set.
- Mulan multi-target regression data sets: Water Quality, EDM, ATP1d, ATP7d,
  and RF1.
- The Linnerud data set bundled with scikit-learn.

All simulation data are generated inside `run_experiments.py` using fixed
random seeds.

## Main generated outputs

- `outputs/raw_results.csv`: fold-level and replicate-level results.
- `outputs/rate_summary.csv`: low-dimensional simulation summary.
- `outputs/scaling_summary.csv`: output-dimension scaling summary.
- `outputs/scaling_speedup_summary.csv`: speedup summary used in the manuscript.
- `outputs/large_scale_stress_summary.csv`: large-scale stress-test timing.
- `outputs/direct_max_sanity_summary.csv`: direct max-feature sanity check.
- `outputs/real_standardized_summary.csv`: real-data six-method summary.
- `outputs/table_*.tex`: LaTeX tables used or referenced by the manuscript.
- `outputs/fig_*.pdf` and `outputs/fig_*.png`: manuscript figures.

## Reproducibility notes

The script fixes random seeds for simulations, train/test splits,
cross-validation folds, random forest baselines, k-means initialization, and
random-center ablations. Runtime values can vary across machines and BLAS
installations, but the reported qualitative comparisons are generated by the
same code path.
